# 📋 Agenda de Contatos

Uma agenda digital de contatos desenvolvida em HTML, CSS e JavaScript puro (Vanilla JS). O projeto foi criado para demonstrar manipulação do DOM, eventos, validação de formulários e gerenciamento de estado na listagem, edição, busca e exclusão de contatos.

## 🔗 Demonstração
O projeto está hospedado e pode ser acessado em: **[https://contatos-projetos.vercel.app/](https://contatos-projetos.vercel.app/)**

## 🚀 Funcionalidades
- **Cadastro de Contatos:** Nome, telefone e e-mail (com validações de formato e e-mail único).
- **Listagem Dinâmica:** Atualização instantânea ao adicionar, editar ou remover.
- **Busca em Tempo Real:** Filtro de contatos instantâneo enquanto você digita.
- **Edição e Remoção:** Botões diretos na lista para gerenciar os dados.

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript (ES6+)
